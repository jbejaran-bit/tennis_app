# Baseline — Tennis Performance Intelligence

Baseline is a premium, AI-powered tennis performance tracking web application. Track matches, visualize performance metrics, and receive professional tactical debriefs driven by Claude AI.

## 🚀 Getting Started

First, install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live app.

## 🎾 Key Features
- **Match Logbook**: Record scorelines, serve rates, error counts, and court surface types.
- **Dynamic Diagnostics**: Automatic tracking of win rates and telemetry trends.
- **Claude AI Coaching**: Tactical recommendations and professional level practice drills generated from post-match statistics.
- **Modern Premium Design**: Dark glassmorphic user interface designed for players who want to elevate their game.
